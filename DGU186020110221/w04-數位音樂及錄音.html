取樣率

Sampling Rate

人的聽覺只能聽到 20 kHz 的聲音，那要用多少頻率來進行取樣呢？

尼奎斯特定律 44.1 kHz

Nyquist's Theorem

> 美國錄音師 Harry Nyquist 認為要獲得優良的類比/數位轉換，需要人類聽覺的2倍速度來取樣。

 

> 2倍是 40 kHz 怎麼會是 44.1 kHz ?

 

> 因為濾波器不可能把 20 kHz 處理乾淨，總會有幾千 Hz 的尾巴，所以要用 44.1 kHz，以防萬一。

 

> 取樣 44.1 kHz 就像每秒對音樂照 44,100 張照片

比特率

Bit rate

聲音取樣另一個決定要素 Bit Rate

 

1 bit : 1表示有訊號 0表示沒有訊號

2 bit : 00 沒訊號 01 有一點 10 大一點 11 最大訊號

3 bit : 000 001 010 011 100 101 110 111

 

常見的 16 bit 表示可表現 65,536 的聲音層次，24 bit 則有 16,777,216。

延遲率 & ASIO

Latency

數位錄音有兩個關念是類比錄音沒有的

1. 電腦記憶體緩衝區大小

2. 監聽延遲時間

 

正常而言 A/D D/A 轉換約 1.5 ms

 

電腦記憶體緩衝區大小 ／ 取樣率 ＝ 監聽延遲時間

EX: Buffer 128 / sampling rate 44100/sec x 1000ms = 2.9 ms

 

機器需要 2.9 ms 處理音樂，所以錄音時音樂由音源發聲到由喇叭監聽所需的時間 1.5ms+1.5ms+2.9ms = 5.9 ms (Total Delay)
